# This repository contains a Financial model and valuation analysis of Infosys Ltd, covering historical performance, ratio analysis, forecasting, DCF-based intrinsic valuation, and cross-check methods such as comparable company and football field analysis. Built for educational and analytical purposes.

## What this model covers

- One-page company profile
- Historical income statement, balance sheet, and cash flow analysis
- Common size financial statements
- Detailed ratio analysis
- Revenue, EBITDA, and EPS forecasting
- Discounted Cash Flow (DCF) valuation
- Sensitivity analysis
- Comparable company valuation
- Football field analysis
- DuPont analysis and Altman Z-score
- Risk analysis using VAR and Monte Carlo simulation

## How to read this model

Start with the one-page company profile, then review historical financial statements and ratio analysis. Move next to the forecasting section, followed by intrinsic valuation using DCF and cross-check valuation methods.

## Key outputs

- Intrinsic value per share derived from DCF analysis
- Valuation range based on comparable company analysis
- Sensitivity of valuation to WACC and terminal growth assumptions

## Skills demonstrated

- Financial statement and ratio analysis
- Forecasting and valuation modeling
- DCF and relative valuation techniques
- Risk analysis using VAR and Monte Carlo simulation

## Disclaimer

This project is created strictly for educational purposes. All assumptions are illustrative and based on publicly available data. This is not investment advice and should not be used as a basis for investment decisions.
